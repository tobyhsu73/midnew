3.2.1 PWM触发说明 3.2.1 PWM Trigger Description
当触发输入引线“RX” 接收到一个高电平触发脉冲，下降沿会触发模组工作一次，输 When the trigger input lead “RX” receives a high-level trigger pulse, the falling edge will trigger the module to work once and lose.
出引线“TX”将输出一次TTL电平的PWM高电平脉宽信号，模组的触发周期必须大于60ms, The lead "TX" will output a PWM high-level pulse width signal of TTL level, and the trigger period of the module must be greater than 60ms.
如果模组没有检测到物体，输出引线“TX”将输出约35ms的固定脉宽。 If the module does not detect an object, the output lead "TX" will output a fixed pulse width of approximately 35ms.
